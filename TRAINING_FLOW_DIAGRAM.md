## ğŸ¯ What Happens in Code

### In `train.py`

```python
for epoch in range(100):  # 100 epochs
    for batch in train_loader:  # 26 batches
        degraded = batch['degraded']  # Step 3
        clean = batch['clean']        # Step 1
        
        restored = model(degraded)    # Step 4
        loss = criterion(restored, clean)  # Step 6
        
        loss.backward()               # Gradients
        optimizer.step()              # Update weights
        scheduler.step()              # Update LR
```
